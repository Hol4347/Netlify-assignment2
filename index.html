Learn more about git and GitHub

Git is the fast and modern implementation of version control. Git can give history of content changes as well as facilitates change to file. Git is an easy to use function for any type of worker. If I can figure it out, odds are you can!

GitHub is an interface that uses Git extensively on the interweb. GitHub is a place to store a code and it can lead to a website being published. You can not use the GitHub service withut using git.

Useful and basic git/GitHub instructions for beginners

    Make sure you create a GitHub account according to your intended purpose of us, either personal or with an organization. Make sure you also install git on the computer you are using, for Mac use terminal and for Windows use Command Prompt.
